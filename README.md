# AI Image Selector 📸

An intelligent system that helps users find the best images from their collection based on their personal preferences.

## Features
- 🎯 **Personalized Ranking** - Learns from images you like
- 🚀 **Fast Processing** - CLIP-based embedding for accurate results
- 📱 **Multi-Platform** - Web, Mobile, and API access
- 🔒 **Privacy First** - Your images stay private
